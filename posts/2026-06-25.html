2. RL Deconvolution + Diffusion Prior — 생성 모델이 역문제 반복 알고리즘 내부로 진입

기존에 diffusion model을 역문제에 쓰는 방식은 주로 '정규화 항으로 활용'하거나 '사후분포 샘플링'이었습니다. 이 논문은 Richardson-Lucy 반복의 각 step에 diffusion prior를 직접 주입하는 방식으로, 생성 모델이 외부 정규화가 아니라 반복 알고리즘 내부 구성요소로 들어오는 패턴을 보여줍니다. PnP MMSE 수렴 이론이 같은 날 나온 것은 이 방향의 이론적 기반도 함께 강화되고 있음을 의미합니다. 형광 현미경을 넘어 FP·ODT·렌즈리스 시스템에서도 같은 패턴을 적용할 수 있습니다.

Confidence: High — arXiv eess.IV + IEEE TCI 두 독립 논문

3. Scattering-Enhanced Tomography — 산란을 역문제의 신호로 재정의

계산 광학 분야에서 산란은 오랫동안 극복해야 할 방해 요소였습니다. 적응광학, 산란 보정 알고리즘, 산란 매질 제거 등이 그 방향의 노력입니다. scattering-enhanced intensity tomography는 이 전제를 뒤집어, 산란 패턴 자체가 3D 물체의 정보를 담고 있다는 역문제 관점을 채택합니다. 즉 더 많이 산란할수록 더 많은 정보를 얻을 수 있다는 역발상입니다. IEEE TCI에 게재됐다는 것은 이 방향이 충분한 이론적·실험적 뒷받침을 갖췄다는 신호입니다.

Confidence: Medium — 단일 IEEE TCI 논문, 독립 검증 필요

🔬 추천 연구주제

A. RL Deconvolution + Diffusion Prior를 FP 시스템에 적용

형광 현미경에서 검증된 RL+diffusion prior 접근을 Fourier ptychography 재구성에 이식합니다. FP는 다중 측정의 위상 복원 문제인데, RL 방식과 구조가 달라 직접 이식보다 원리 차용이 현실적입니다.

실행 1주차:

[TIER A] Improving Richardson-Lucy Deconvolution with Diffusion Priors for Fluorescence Microscopy (2606.25924)

[방법전환] [실사용전환]

왜 A급인가: 형광 현미경 역문제에서 생성 모델이 외부 정규화가 아니라 반복 알고리즘 내부에 통합되는 패턴을 처음 보여준 논문입니다. RL deconvolution은 광학 현미경의 기본 도구인데, diffusion prior를 각 이터레이션에 주입함으로써 포아송 노이즈 조건에서도 수렴 품질을 개선합니다. 이 접근은 FP·ODT·렌즈리스 시스템으로 이식 가능한 패턴으로, 랩 파이프라인에 직접 영향을 줍니다.

핵심: Richardson-Lucy는 최대우도 추정(포아송 데이터)으로 반복을 돌리는데, 각 반복 후 diffusion denoiser를 적용하는 방식입니다. 이는 PnP와 비슷하지만, RL의 포아송 데이터 피델리티를 유지하면서 diffusion 사전 분포를 통합한다는 점이 다릅니다.

주의: 형광 현미경 PSF 가정이 FP·ODT 설정과 다를 수 있으므로 이식 시 forward model 재정의 필요. 본문의 ablation 결과와 실험 조건 확인 필요.

[TIER A] Scattering-enhanced intensity based tomography — IEEE Transactions on Computational Imaging

[방법전환] [문제정의]

왜 A급인가: 산란을 역문제의 장애물이 아니라 신호원으로 재정의하는 논문입니다. IEEE Transactions on Computational Imaging에 게재됐다는 것은 이론적·실험적 뒷받침이 충분하다는 신호입니다. 계산 광학 역문제의 기본 가정을 뒤집는 논문이므로, 고산란 생물 샘플에서 ODT를 사용하는 랩에 직접 영향을 줍니다.

핵심: 산란이 많은 매질에서 강도(intensity) 측정값이 산란에 의해 추가 정보를 포함한다는 점을 역문제로 활용. 기존 ODT가 '산란 → 잡음'으로 모델링하는 것과 반대입니다.

주의: 어느 산란 레짐(μs' 범위)에서 적용 가능한지, 필요한 추가 측정이 있는지 본문 확인 필요.
